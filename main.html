Why I want to become a developer?

One of the most commonly opted engineering streams by the students is Software Engineering. Software Engineering is one of the branches of engineering which deals with designing, writing, developing and maintaining software programs for computers and other devices operated electronically. The primary job of a software engineer is to write a software program or modify existing software programs in order to improve the program and make it more efficient than before. Some of the requirements by software engineering are to be able to scale up the software as per the changes that are required. The software that has been programmed should be correct as per the requirement of the clients. The software should be dynamic enough to be able to rehashed and use for making a different product.
